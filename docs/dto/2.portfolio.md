# PortfolioItem

포트폴리오 아이템 정보. 목록 화면에 표시되는 기본 정보.

## 계층 구조

```
PortfolioCategory (카테고리)
  └── PortfolioItem (아이템 목록)  ← 현재 위치
        └── PortfolioItemDetail (상세 정보)
```

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `id` | string | O | 아이템 고유 ID |
| `categoryId` | string | O | 소속 카테고리 ID |
| `title` | string | O | 아이템 제목 |
| `summary` | string | O | 한 줄 요약 (목록에서 표시) |
| `thumbnailUrl` | string | X | 썸네일 이미지 URL |
| `tags` | string[] | O | 분류 태그 목록 |
| `createdAt` | string | O | 생성일 (YYYY-MM-DD) |
| `order` | number | O | 정렬 순서 |

## TypeScript Interface

```typescript
export interface PortfolioItem {
  id: string;
  categoryId: string;
  title: string;
  summary: string;
  thumbnailUrl?: string;
  tags: string[];
  createdAt: string;
  order: number;
}
```
