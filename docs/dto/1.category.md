# PortfolioCategory

포트폴리오 카테고리(그룹) 정보. 최상위 분류 단위.

## 계층 구조

```
PortfolioCategory (카테고리)
  └── PortfolioItem (아이템 목록)
        └── PortfolioItemDetail (상세 정보)
```

## Fields

| Field         | Type   | Required | Description                              |
| ------------- | ------ | -------- | ---------------------------------------- |
| `id`          | string | O        | 카테고리 고유 ID                         |
| `name`        | string | O        | 카테고리 이름 (예: "게임", "앱", "그림") |
| `description` | string | O        | 카테고리 설명                            |
| `imageUrl`    | string | X        | 대표 이미지 URL                          |
| `order`       | number | O        | 정렬 순서                                |

## TypeScript Interface

```typescript
export interface PortfolioCategory {
  id: string;
  name: string;
  description: string;
  imageUrl?: string;
  order: number;
}
```
