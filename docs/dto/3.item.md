# PortfolioItemDetail

포트폴리오 아이템의 상세 정보. 상세 페이지에서 표시되는 추가 정보.

## 계층 구조

```
PortfolioCategory (카테고리)
  └── PortfolioItem (아이템 목록)
        └── PortfolioItemDetail (상세 정보)  ← 현재 위치
```

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `itemId` | string | O | 연결된 PortfolioItem ID |
| `description` | string | O | 상세 설명 (마크다운 지원) |
| `images` | Image[] | X | 이미지/스크린샷 목록 |
| `links` | Link[] | X | 관련 링크 목록 |
| `highlights` | string[] | X | 주요 특징/기능 목록 |
| `metadata` | object | X | 카테고리별 추가 메타데이터 |

## TypeScript Interface

```typescript
export interface PortfolioItemDetail {
  itemId: string;
  description: string;
  images?: Image[];
  links?: Link[];
  highlights?: string[];
  metadata?: Record<string, unknown>;
}

export interface Image {
  url: string;
  caption?: string;
  type?: 'screenshot' | 'photo' | 'artwork';
}

export interface Link {
  url: string;
  label: string;
  type: 'github' | 'demo' | 'download' | 'blog' | 'store' | 'other';
}
```
