Summary

핵심 요약

  • portfolio_project_client_vite는 Portfolio 관리용 React SPA입니다.
  • portfolio_project_server_flask는 Flask 기반 REST API 서버입니다.
  • 클라이언트와 서버는 /api/v1 REST API와 JWT access/refresh token으로 연동합니다.
  • 상위 묶음 도메인은 Portfolio, 실제 작업물 도메인은 Project로 통일했습니다.
  • 서버 문서와 실행 경로는 Flask 구현에 맞춰 portfolio_project_server_flask를 기준으로 작성합니다.
