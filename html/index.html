Summary

핵심 요약

  • portfolio_project_client_vite는 포트폴리오 관리용 React SPA입니다.
  • portfolio_project_server_fastapi는 이름과 달리 실제 구현이 FastAPI가 아닌 Flask 기반입니다.
  • 클라이언트와 서버는 /api/v1 REST API와 JWT access/refresh token으로 연결됩니다.
  • 도메인상 사용자는 카테고리 역할의 Portfolio를 만들고, 그 안에 여러 Project를 등록합니다.
  • 카테고리와 프로젝트의 공개 여부에 따라 공개 URL에서 노출되는 데이터가 결정됩니다.
명칭 주의: 서버 디렉터리명과 README 제목은 FastAPI처럼 보이지만, 코드와 의존성은 Flask 기반입니다.
