Run

실행 방법

프로젝트 루트에서 아래 명령을 실행하면 프론트엔드와 백엔드가 함께 시작됩니다.

npm run dev
  • 프론트엔드는 portfolio_project_client_vite에서 Vite 개발 서버로 실행됩니다.
  • 백엔드는 portfolio_project_server_flask에서 .venv를 활성화한 뒤 python app.py로 실행됩니다.
  • 기본 주소는 프론트엔드 http://localhost:5173, 백엔드 http://localhost:8000입니다.
  • Swagger UI는 개발 환경에서 http://localhost:8000/docs로 확인할 수 있습니다.
Current Snapshot

현재 코드 상태 요약

  • portfolio_project_client_vite는 React SPA입니다. 보호 라우트와 공개 라우트를 분리하고, 서버의 /api/v1 API를 호출합니다.
  • portfolio_project_server_flask는 이름 그대로 Flask 기반 REST API 서버입니다. 예전 FastAPI 명칭은 현재 코드와 맞지 않습니다.
  • 상위 묶음 도메인은 Portfolio, 개별 작업물 도메인은 Project로 통일되어 있습니다.
  • 클라이언트는 공통 UI 컴포넌트와 페이지 로직 훅을 사용해 렌더링과 상태/API 로직을 분리했습니다.
  • 서버 Swagger UI는 개발 환경에서 http://localhost:8000/docs로 제공됩니다.
