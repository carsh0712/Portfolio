SEO와 공유 미리보기

공개 Portfolio와 Project URL이 검색엔진과 카카오톡 같은 메신저에서 더 잘 읽히도록 적용한 HTML 메타 처리 기준입니다.

Goal

왜 서버에서 처리했나

공개 페이지는 React SPA로 렌더링되지만, 카카오톡이나 검색엔진 크롤러는 JavaScript 실행 전의 HTML만 읽는 경우가 많습니다. 그래서 /public/:username/:portfolioCode와 /public/:username/:portfolioCode/:projectCode 요청에 대해 Flask가 index.html을 읽고 페이지별 메타 태그를 먼저 주입하도록 구성했습니다.

결과: 브라우저 사용자는 기존 SPA를 그대로 보고, 공유 봇은 HTML 응답 안의 title, description, Open Graph 이미지를 읽을 수 있습니다.
Meta Tags

주입하는 HTML 메타

공개 Portfolio와 Project 페이지에는 다음 정보를 HTML <head> 안에 추가합니다.

<title>페이지 제목</title>
<link rel="canonical" href="공개 URL" />
<meta name="description" content="페이지 설명" />
<meta property="og:title" content="페이지 제목" />
<meta property="og:description" content="페이지 설명" />
<meta property="og:url" content="공개 URL" />
<meta property="og:type" content="website" />
<meta property="og:image" content="대표 이미지 URL" />
<meta name="twitter:card" content="summary_large_image" />
  • Portfolio URL은 Portfolio 이름과 설명을 사용합니다.
  • Project URL은 Project 제목과 요약을 우선 사용합니다.
  • 대표 이미지는 Portfolio 대표 이미지, Project 썸네일, 첫 공개 Project 이미지 순서로 선택합니다.
  • 이미지가 없으면 이미지 메타는 생략하고 title/description 중심으로 노출합니다.
Implementation

서버 처리 흐름

1. 공개 URL 요청이 Flask로 들어옵니다.
2. 서버가 사용자, Portfolio, Project 공개 여부를 확인합니다.
3. portfolio_project_client_vite/dist/index.html을 UTF-8로 읽습니다.
4. 기존 기본 <title>을 제거하고 페이지별 메타 태그를 삽입합니다.
5. 원래의 <div id="root">와 Vite script는 유지해 React 앱이 그대로 실행되게 합니다.

구현 위치는 portfolio_project_server_flask/app.py입니다. 테스트는 portfolio_project_server_flask/tests/test_client_static.py에서 메타 태그와 SPA 루트 보존 여부를 확인합니다.

Default HTML

기본 index.html fallback

공개 상세 URL처럼 서버가 동적으로 메타를 주입하는 경로가 아닌 일반 진입점도 최소한의 설명을 갖도록 Vite 기본 index.html에 description, Open Graph, Twitter card 기본값을 추가했습니다.

주의: 기본 index.html의 메타는 fallback입니다. 공개 Portfolio/Project 공유 품질은 서버 주입 메타가 우선 담당합니다.
Verification

확인 방법

  • 로컬에서는 HTML 응답의 <head>에 메타 태그가 들어갔는지 확인합니다.
  • 카카오톡 미리보기는 공개 HTTPS URL 배포 후 Kakao Developers 공유 디버거에서 확인합니다.
  • 카카오톡은 미리보기 캐시가 강하므로, 수정 후에는 공유 디버거에서 캐시를 갱신하거나 URL query를 바꿔 테스트합니다.
  • 이미지가 나오지 않으면 og:image URL이 외부에서 인증 없이 접근 가능한지, 이미지 형식과 크기가 적절한지 확인합니다.
