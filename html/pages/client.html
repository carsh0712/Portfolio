Client

개요

역할

React 기반 포트폴리오 관리 SPA입니다. 로그인 후 카테고리와 프로젝트를 생성/수정하고, 공개 URL로 프로젝트를 노출합니다.

기술 스택

  • React 19, react-dom
  • react-router-dom 7
  • Vite 7, @vitejs/plugin-react-swc
  • Tailwind CSS 4
  • Vitest, Testing Library, MSW, Playwright
Structure

디렉터리 구조

경로 역할
src/App.tsx전체 React Router 구성. 인증 페이지, 공개 페이지, 보호된 관리 페이지를 선언합니다.
src/contexts/AuthContext.tsx로그인, 로그아웃, 토큰 저장, 사용자 정보 상태를 관리합니다.

src/pages로그인, 회원가입, 카테고리 목록/생성/수정, 프로젝트 목록/생성/상세, 공개 목록/상세 화면입니다.
src/componentsHeader, ProtectedRoute, 폼, 카드, 모달, 이미지 미리보기, SVG 아이콘 등 재사용 UI 컴포넌트입니다.
src/types서버 요청/응답에 대응하는 TypeScript 타입입니다.
src/test, *.test.tsxVitest, Testing Library, MSW 기반 테스트입니다.
e2e/example.spec.tsPlaywright E2E 테스트 진입점입니다.

Auth

인증 상태 관리

  • AuthProvider가 auth_tokens와 auth_user를 localStorage에 저장합니다.
  • 로그인 성공 후 /api/v1/user/me를 호출해 사용자 정보를 가져오고 /home으로 이동합니다.
  • ProtectedRoute는 isAuthenticated가 거짓이면 /login으로 리다이렉트합니다.
  • 로그아웃은 로컬 상태를 먼저 지우고, refresh token이 있으면 백그라운드로 /api/v1/auth/logout을 호출합니다.

Runbook

실행과 테스트

cd portfolio_project_client_vite
npm install
npm run dev
npm test
npm run test:e2e

API 주소는 VITE_API_BASE_URL 환경변수로 지정할 수 있고, 없으면 http://localhost:8000을 사용합니다.
