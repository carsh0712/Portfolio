Client

개요

역할

React 기반 Portfolio 관리 SPA입니다. 로그인 후 Portfolio와 Project를 생성, 수정하고 공개 URL로 Project를 노출합니다.

기술 스택

  • React 19, react-router-dom 7
  • Vite 7, @vitejs/plugin-react-swc
  • Tailwind CSS 4
  • Vitest, Testing Library, MSW, Playwright
Structure

디렉터리 구조

경로역할
src/App.tsxReact Router 구성입니다. 인증 페이지, 공개 페이지, 보호된 관리 페이지를 선언합니다.
src/pagesPortfolioList, PortfolioAdd, PortfolioEdit, ProjectList, ProjectAdd, ProjectDetail 등 화면 단위 컴포넌트입니다.
src/componentsPortfolioForm, ProjectEditForm, 카드, 모달, 이미지 미리보기, SVG 아이콘 등 재사용 UI입니다.
src/typesportfolio.ts와 project.ts로 Portfolio와 Project 타입을 분리합니다.
src/utils/api.ts서버 API 호출 계층입니다. 인증 헤더 추가와 refresh token 재시도 로직을 포함합니다.
src/test, *.test.tsxMSW 기반 API mock과 컴포넌트, 유틸 테스트입니다.
Routes

화면 라우트

URL컴포넌트인증설명
/, /loginLogin불필요이메일과 비밀번호로 로그인합니다.
/registerRegister불필요username, email, password로 회원가입합니다.
/homePortfolioList필수로그인 사용자의 Portfolio 목록입니다.
/portfolio/addPortfolioAdd필수새 Portfolio를 생성합니다.
/portfolio/:portfolioCode/editPortfolioEdit필수Portfolio 이름, 설명, 대표 이미지, 공개 여부를 수정합니다.
/portfolio/:portfolioCodeProjectList필수Portfolio 안의 Project 목록을 조회하고 태그로 검색합니다.
/portfolio/:portfolioCode/project/addProjectAdd필수새 Project를 등록합니다.
/portfolio/:portfolioCode/project/:projectCodeProjectDetail필수Project 상세를 조회하고 수정합니다.
/public/:username/:portfolioCodePublicPortfolioList공개공개 Portfolio의 Project 목록입니다.
/public/:username/:portfolioCode/:projectCodePublicProjectDetail공개공개 Project 상세 페이지입니다.
API

클라이언트 API 함수

함수서버 엔드포인트용도
signupPOST /api/v1/auth/signup회원가입
getCurrentUserGET /api/v1/user/me현재 사용자 조회
getPortfoliosGET /api/v1/portfolios/Portfolio 목록 조회
createPortfolioPOST /api/v1/portfolios/Portfolio 생성
getPortfolioDetailGET /api/v1/portfolios/{code}Portfolio 상세 조회
updatePortfolioPUT /api/v1/portfolios/{code}Portfolio 수정
deletePortfolioDELETE /api/v1/portfolios/{code}Portfolio 삭제
getProjectsGET /api/v1/projects/Portfolio별 Project 목록 조회
getProjectDetailGET /api/v1/projects/{portfolioCode}/{projectCode}Project 상세 조회
createProjectPOST /api/v1/projects/Project 생성
updateProjectPUT /api/v1/projects/{portfolioCode}/{projectCode}Project 수정
getPublicProjectsGET /api/v1/public/{username}/{portfolioCode}/공개 Project 목록
getPublicProjectDetailGET /api/v1/public/{username}/{portfolioCode}/{projectCode}/공개 Project 상세
uploadImagePOST /api/v1/files/upload이미지와 파일 업로드
Runbook

실행과 테스트

cd portfolio_project_client_vite
npm install
npm run dev
npm test
npm run test:e2e

API 주소는 VITE_API_BASE_URL 환경 변수로 지정하고, 없으면 http://localhost:8000을 사용합니다.
